# Setup Axios

This plugin gives you useful functions which is extending the `Repository`

## Install Pinia ORM Axios Plugin

::code-group
  ```bash [Yarn]
  yarn add @pinia-orm/axios
  ```
  ```bash [NPM]
  npm install @pinia-orm/axios --save
  ```
  ```bash [PNPM]
  pnpm add @pinia-orm/axios
  ```
::
