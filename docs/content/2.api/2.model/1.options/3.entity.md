# `entity`

::alert{type='warning'}
This field is required and must be set in every model
::

## Usage

````js
class User extends Model {
  static entity = 'users'

  static fields () {
    return {
      userId: this.attr(null)
    }
  }
}
````

## Typescript Declarations
````ts
const entity: string = undefined
````
